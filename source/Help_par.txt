Help for parameters

Number of parameters: Add the number of parameters in FDS input file (PAR) and press 'Enter' 

Define the parameter types:
	t(T_MAX) - the final time of temperature ramp (&RAMP T): Fill the maximum (and initial) temperature of the ramp.
	TWFIN - the final time of simulation (same as the final time of temperature ramp)
	NU_FUEL - Mass fraction of fuel gases, 1-nu_residue: Choose the corresponding residue variable.
	Ramp_value - Actually not only for ramp values. This parameter has value that is equal or greater than the corresponding variable.
			User has to define the corresponding variable, the upper boundary for this value and if log-scale is used.

Finish by clicking 'OK'
